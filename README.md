# Náš (Barči & Vojty) svatební web

Tento svatební web slouží k předání informací mezi námi a svatebčany (hosty). Webová stránka obsahuje základní informace (např. datum a místo obřadu) a také formulář k vyplnění hosty, kteří v něm uvedou např. preferovanou variantu jídla.

## Architektura webu

Frontend svatebního webu je hostován na GitHub Pages. Web obsahuje formulář, po jehož vyplnění a odeslání se spustí backendová aplikace (Google Apps Script), která zapíše vyplněné informace do Tabulky Google.
